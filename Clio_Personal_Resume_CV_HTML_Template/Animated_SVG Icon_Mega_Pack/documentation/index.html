Introduction

Bored with traditional hover effects? Want something more lively to stand out your business site/project? This incredible super animated icon set has it all! Please take a few second to watch the video demo upward & get ready to be amazed!

Features

  • 120 Icons
  • Code Snippets (Click & Copy the codes, save time)
  • Ready animations
  • 5+ animation variations to play with
  • Perfectly Scalable (non destructive)
  • Compatible with all major browsers
  • Highly customizable (color, size, stroke)
  • Rich documentations
  • Themecop standard support
